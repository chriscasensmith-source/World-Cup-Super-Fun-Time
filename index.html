📖 How It Works

🐍 Draft Rules

  • 48 World Cup teams available; only 36 get drafted.
  • 3 owners, 12 teams each, snake order.
  • Order reverses every round (1‑2‑3, 3‑2‑1, …).
  • Each pick is locked to the owner on the clock.
  • Commissioner can undo, move, or reset before lock.

🔒 Lock & Publish

  • Draft on one device (or sync across phones) — picks save automatically.
  • When done, hit Export Lock JSON on the Draft tab.
  • Replace public/data/draft-lock.json with it and push.
  • Once "locked": true, every visitor sees the same frozen draft.
  • Live scores update from the football-data.org GitHub Action.
